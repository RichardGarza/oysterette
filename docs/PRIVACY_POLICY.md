## Data Storage and Security

### Storage Locations

**Local Storage (On Your Device):**
- Authentication token (encrypted)
- User preferences (theme, settings)
- Favorite oysters list (synced with server)
- Cached app data

**Server Storage (Secure Cloud Database):**
- Account information
- Reviews and ratings
- Favorites
- User statistics
- Hosted on Neon (PostgreSQL) with encryption at rest

**Backend Server:**
- Hosted on Railway with HTTPS encryption
- Located in United States
- SOC 2 compliant infrastructure

### Security Measures

We implement industry-standard security measures:
- **Encryption in Transit:** All data transmitted using HTTPS/TLS
- **Password Security:** Passwords hashed using bcrypt (industry standard)
- **Token-Based Authentication:** JWT tokens with expiration
- **Rate Limiting:** Protection against brute-force attacks
- **Input Validation:** Prevention of SQL injection and XSS attacks
- **Secure Storage:** AsyncStorage for local data (encrypted by OS)

### Data Retention

- **Active Accounts:** Data retained while account is active
- **Deleted Accounts:** All personal data deleted within 30 days
- **Reviews:** May be anonymized instead of deleted to preserve community ratings
- **Backups:** Retained for 30 days for disaster recovery
